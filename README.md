## What I Build

Open-source networking and privacy tools — from DNS management desktops to proxy panels and config generators.

### SparkDns
Cross-platform DNS management desktop app built with **Tauri 2** (Rust + TypeScript). One-click DNS switching, diagnostics, and system tools for Windows, Linux, and macOS.

`rust` `typescript` `tauri` `dns` `cross-platform`

[![Release](https://img.shields.io/badge/release-v1.0.0-4ec9b0?style=flat-square)](https://github.com/Code-Leafy/SparkDns/releases/latest)

</td>
<td width="50%" valign="top">

### G2Leafy
Web dashboard for managing **Xray VLESS xHTTP** configs on GitHub Codespaces. Client management, subscription generation, live analytics — all from a browser.

`python` `xray` `vless` `xhttp` `web-panel`

[![Status](https://img.shields.io/badge/status-stable-2DC94E?style=flat-square)](https://github.com/Code-Leafy/G2Leafy)

</td>

### NetLeafy
Client-side **VLESS-over-xHTTP** configuration generator. Bypass network restrictions using Netlify's edge network — 100% serverless, no data leaves your browser.

`javascript` `vless` `xhttp` `netlify` `bypass`

[![Live](https://img.shields.io/badge/try-it-007acc?style=flat-square)](https://code-leafy.github.io/NetLeafy)

</td>
<td width="50%" valign="top">

### NetLeafyScanner
Massively parallel **IP/SNI discovery** and real-proxy speed tester. Tests actual bandwidth through xray-core for VLESS & Trojan configs.

`python` `scanner` `xray` `speed-test` `networking`

[![Status](https://img.shields.io/badge/status-active-459494?style=flat-square)](https://github.com/Code-Leafy/NetLeafyScanner)

</td>
</tr>

### SshTunnel
One-command **SOCKS5 proxy** panel for any VPS. Deploys a hardened Docker container with aggressive TCP tuning (BBR, fq, TFO) and a live Textual dashboard.

`python` `socks5` `docker` `tcp-tuning` `vps`

[![Status](https://img.shields.io/badge/status-active-4ec9b0?style=flat-square)](https://github.com/Code-Leafy/SshTunnel)
